# USER.md — Perfil del usuario

## Proyecto
- **Nombre**: KRIPTONSHARE
- **Stack**: Flutter + Supabase + AES-256-GCM
- **Plataforma objetivo**: Android (MVP), iOS/web futuro

## Preferencias detectadas
- **Idioma de trabajo**: español
- **Flujo prioritario**: compartir archivos cifrados entre usuarios registrados mediante links `/room/<id>`
- **Seguridad**: ambos usuarios deben tener cuenta en KRIPTONSHARE; la contraseña nunca se almacena
- **Experiencia deseada**: el receptor debe ver claramente que recibió un archivo y poder abrirlo dentro de la app

## Notas
- Usa Windows como entorno de desarrollo local.
- Pruebas E2E con dos dispositivos Android físicos.
